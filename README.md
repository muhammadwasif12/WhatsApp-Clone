## ğŸš€ Getting Started

### Prerequisites
- Flutter SDK (>=3.0.0)
- Dart SDK (>=3.0.0)
- Android Studio / Xcode (for emulator/device)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/muhammadwasif12/WhatsApp-Clone.git
   cd WhatsApp-Clone
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
